A member function is a function of a class that is applied to an instance of that class, i.e. an object, using the dot notation. For example, the statement wakeup_time.add_seconds(10000); is an application of the member function add_seconds to the Time object called wakeup_time. Member functions are not allowed to be used by themselves, they must always be applied to an object. A nonmember function is not a member of any class, hence it has no implicit parameter. Nonmember functions are not applied to objects.
