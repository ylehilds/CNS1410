Once a day, carry out the following: 
a. 	Format a floppy and label it with the day of the week: Monday, Tuesday, .., Friday.
b.  Using Explorer, select the folder c:\work\programs and drag it to the floppy drive 
c.  Remove the floppy and put it in a safe place.

Once a week, verify the backup by using the following procedure.
a.  Using Explorer, make a new folder Test
b.  Insert the latest backup in the floppy drive
c.  Using Explorer, drag the floppy into the Test folder.
d.  Count the number of files in the Test folder and the c:\work\programs folder. Is the count different?
e.  Open one of the files with extension .cpp at random in the Test folder. Pick the corresponding file in the c:\work\programs folder. Do they look different? 
f.  If neither d. nor e. showed a problem, the verification passed.
